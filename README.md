# AxReact 

使用 webpack + react + redux + es6 的组件化前端项目 AxReact。

因为最近在工作中尝试了 [webpack](https://github.com/webpack/webpack)、[react](https://github.com/facebook/react)、[redux](https://github.com/reactjs/redux)、[es6](http://babeljs.io/docs/learn-es2015/) 技术栈，所以总结出了一套 [AxReact]，以便下次做项目时可以快速开始，并进行持续优化。

> 开发时的环境

- [x] Mac OSX Sierra
- [x] Node v8.0.0
- [x] NPM v5.2.0

> 快速开始

```
$ npm install
$ npm start
```
